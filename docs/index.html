Wokshop: A Practical Introduction to Debugging

Description

This workshop is designed for beginners. We will introduce how to approach debugging computer programs using the command line tool gdb and provide interactive examples to get you started. We will also introduce you to memory debugging with valgrind and discuss options for debugging computer programs that utilize parallel processing.

Requirement

  1. To install the required software, please follow the instructions at the bottom of the page.
  2. Some prior knowledge of coding with C/C++/Fortran is recommended but not requitred.
